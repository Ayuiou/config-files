git init -инициализация репозитория

git status- проверить статус файлов

git add . - добавить все файлы в область видимости git

git commit- сохранить изменения

git push origin <имя коммита> - закинуть все на github

git pull- взять все файлы из репозитория и обновить их

git clone ссылка - копировать все файлы на компьютер с github
